# CS181_Final_Project

## Data

All project datasets are stored in `data/`.

- `curated_seed_500.csv`: manually curated seed examples.
- `deepseek_generated_3000.csv`: final balanced DeepSeek-generated dataset.
- `train.csv`, `val.csv`, `test.csv`: train/validation/test splits derived from the generated dataset.
- `split_manifest.json`: metadata for the split files.
